# Voxel Game v1.0.0

Initial portfolio-ready desktop release.

## Highlights

- Runnable desktop Java/LWJGL build packaged as a fat JAR.
- Procedural voxel terrain with biomes, terrain height variation, ores, water, trees, and decorative vegetation.
- First-person block interaction with mining, placement, inventory, hotbar, crafting, furnaces, tools, food, health, and hunger.
- Passive animal framework with AI foundations, pathfinding basics, damage, death, item drops, and persistence.
- Village and villager foundations with generated villages, homes, schedules, professions, trading hooks, and test-friendly village spawning.
- Decorative block expansion with plants, stairs, fences, gates, doors, beds, lamps, lanterns, and bells.
- GitHub Pages inspection site and source archive packaging for CV sharing.

## Artifacts

- `voxel-game.jar` - runnable desktop build.
- `voxel-game-1.0.0-source.zip` - source, assets, docs, scripts, and Gradle project files.

## Run

```powershell
java --enable-native-access=ALL-UNNAMED -jar voxel-game.jar
```

## Web Playability

This release is desktop-only. The game uses LWJGL, GLFW, OpenGL, and native platform libraries, so it is not directly hostable as a browser game on GitHub Pages. The `/docs` page is intended as the live inspection/portfolio preview.
